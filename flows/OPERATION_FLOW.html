üìÅ Core Files Summary

backend/main.py - FastAPI application, endpoints (upload, chat, health, clear), initialization
backend/ingest.py - Document ingestion, extraction (PDF, DOCX, CSV, Excel, PPTX, HTML, XML, Images), chunking
backend/vectorstore.py - FAISS vector store operations (add_chunks, search, clear, save_index)
backend/rag_engine.py - RAG query processing (answer_query_with_context, _build_prompt)
backend/llm_loader.py - LLM engine initialization and generation (GroqLLMEngine)
backend/config.py - Configuration management (Settings class)
backend/logger_config.py - Logging setup (setup_logger)
logs/rag_system.log - All flow logs are written here with rotation
